# 🎓 Grade — выбирай курсы с умом

**Grade** — это аналитическая платформа для выбора онлайн-курсов без маркетингового шума.  
Мы показываем не обещания, а факты: стоимость, содержание, отзывы и реальную пользу обучения.

> Учёба без шума. Только факты.

---

## 🧠 Идея проекта

Современный рынок онлайн-образования перегружен:
- громкими заголовками  
- «успешными кейсами»  
- обещаниями быстрых результатов  

Grade создан как альтернатива витрине маркетинга.  
Проект помогает принимать решения на основе данных, а не эмоций.

---

## ✨ Основные возможности

- 📊 Анализ онлайн-курсов по объективным параметрам  
- ⚖️ Сравнение программ, а не рекламных формулировок  
- 🧩 Подбор курсов под реальные цели пользователя  
- 🔍 Прозрачность: плюсы, минусы и подводные камни  

---

## 🖥️ Технологии
