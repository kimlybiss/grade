# GRADE' — Сервис анализа онлайн-образования

**GRADE'** — это веб-платформа для осознанного выбора курсов. В отличие от агрегаторов-витрин, проект ориентирован на честный анализ программ, сравнение доли практики и реальной ценности обучения.
